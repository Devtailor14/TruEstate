# Retail Sales Management System

## Overview
A full-stack Retail Sales Management System consisting of a Node.js/Express backend and a React/Vite frontend. Designed to demonstrate advanced search, filtering, sorting, and pagination capabilities.

## Tech Stack
- **Backend**: Node.js, Express.js, In-Memory DB
- **Frontend**: React, Vite, TailwindCSS
- **Tools**: npm, Git

## Setup
1. **Backend**:
    ```bash
    cd backend
    npm install
    npm run dev
    ```
2. **Frontend**:
    ```bash
    cd frontend
    npm install
    npm run dev
    ```

For detailed architecture, see [docs/architecture.md](docs/architecture.md).
